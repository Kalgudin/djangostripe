Необходимые настройки для работы.

1. Зарегистрируйте учетную запись Stripe (https://dashboard.stripe.com/register) и перейдите на панель мониторинга. Нажмите "Developers"
2. Затем на левой боковой панели нажмите "API keys". Установите свои секретные ключи в файле settings.py
3. Вам нужно будет указать "Имя учетной записи" в "Настройках учетной записи" по адресу https://dashboard.stripe.com/settings/account.
4. Запускаем удаленный сервер, нажимаем на кнопку покупки товара попадаем на страничку Stripe для оплаты. Заполняем форму(номер тестовой карты 4242 4242 4242 4242)
5. Проверить платежи можно по ссылке: https://dashboard.stripe.com/test/payments
